# 🚀 Detecção de Anomalias em Transações com Python

## 📋 Sobre o Projeto

Este projeto foi desenvolvido durante o Bootcamp Accenture - Python para Análise e Automação de Dados, promovido pela DIO.

O objetivo é identificar transações anômalas utilizando técnicas de análise de dados e aprendizado de máquina, contribuindo para a detecção de possíveis fraudes e comportamentos atípicos em ambientes financeiros.

---

## 🎯 Objetivos

- Realizar análise exploratória dos dados;
- Identificar padrões de comportamento;
- Detectar anomalias em transações;
- Aplicar técnicas de Machine Learning;
- Avaliar os resultados obtidos.

---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

---

## 📊 Etapas do Projeto

### 1. Importação das Bibliotecas
Carregamento das bibliotecas necessárias para análise e modelagem.

### 2. Carregamento dos Dados
Leitura e preparação da base de dados.

### 3. Análise Exploratória
Visualização e compreensão dos dados.

### 4. Tratamento dos Dados
Limpeza e preparação para modelagem.

### 5. Detecção de Anomalias
Aplicação dos algoritmos para identificação de transações suspeitas.

### 6. Avaliação dos Resultados
Análise do desempenho do modelo.

---

## 📈 Resultados

O modelo foi capaz de identificar padrões fora do comportamento esperado, permitindo destacar transações potencialmente anômalas para análise posterior.
